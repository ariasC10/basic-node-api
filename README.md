# Basic REST API with Nodejs
Api built it with Express.js to understand the framework capabilities, as well their behaviour with a noSql DB like mongo.
___
### Packages version
- **express:** 4.18.2
- **mongoose:** 8.0.1
- **body-parser:** 1.20.2

___
### HTTP actions
- Retrieve a list of items (GET)
- Create an item (POST)
